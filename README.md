# trivia
## Trivia Game
My take on a trivia game. Done with a full SDLC proof of concept from a class and sequence UML diagram I thought of, I used flutter framework,
dart code, HTML, CSS and paired programming with codex to get a full working base up. Along with a RESTAPI from https://opentdb.com, and supabase
as a SQL database. The app is up and fully functional.

### Base Gameplay (February 20, 2026)
Sign up and create a user account
Click create a game to search for a friend to challenge, or find someone at random
Pick your 5 categories you think you would win at
Once the game is created your opponent will have to accept and then the game will be generated
You have 3 rounds to score the most points. The first two being 10 multiple choice questions and the last being a fill in the blank
The highest score at the end of the 3 rounds wins!
